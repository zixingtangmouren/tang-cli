# tang-cli

一个功能齐全的脚手架。支持`Vue, React，Angular`框架的项目模板构建。还可以选择移动端和 pc 端模板。

## 快速上手

1. 现在脚手架的 npm 包

```hash
npm install tangmr-cli -g
```

2. 使用 tang-cli

```hash
tang create demo-app
```
