# 计划表

## v2 版本

1. 支持框架（React, Vue2）选择
2. 支持 feature 配置
3. 支持细化的 feature
4. 内部集成`babel, eslint, webpack, css Pre-Processors, Router, Store`
5. 生成项目静态资源

## v3 版本

1. 支持 Vue3 框架选择
2. 支持移动端，Pc 端项目模板选择
3. 支持 npm,yarn 的检测功能
4. 支持自动下载依赖功能
5. 优化原有的工具类
